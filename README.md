# Telegram Mini App для записи в салон

Демо-проект для портфолио: онлайн-запись клиентов в салон красоты через Telegram Mini App.

## Живое демо

- GitHub Pages: https://emoread23.github.io/project134/
- Быстрый просмотр: https://raw.githack.com/emoread23/project134/main/index.html

## Что показывает демо

- выбор услуги;
- выбор мастера;
- выбор даты и времени;
- ввод имени и телефона;
- подтверждение записи;
- понятная витрина для клиента из бьюти-сферы.

## Как продавать клиенту

Можно писать салонам так:

> Я делаю Telegram-бота для онлайн-записи: клиент сам выбирает услугу, мастера и время, а администратор получает готовую заявку. Вот демо, как это выглядит: https://emoread23.github.io/project134/

## Что можно добавить в реальный проект

- Telegram Bot API;
- база данных записей;
- админка для услуг, мастеров и графика;
- уведомления клиенту и администратору;
- интеграция с Google Calendar.

## Стек демо

Статический HTML/CSS/JavaScript. Это сделано специально, чтобы клиент мог открыть пример без установки.
