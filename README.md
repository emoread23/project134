# Atelier Mira — Telegram Mini App для записи в салон

Кейс для портфолио: онлайн-запись клиентов в салон красоты через Telegram Mini App.

## Живой проект

- Открыть проект: https://emoread23.github.io/project134/
- GitHub Pages после включения в настройках: https://emoread23.github.io/project134/

## Что реализовано

- выбор услуги;
- выбор мастера;
- выбор даты и времени;
- ввод имени и телефона;
- подтверждение записи;
- понятная витрина для клиента из бьюти-сферы.

## Как продавать клиенту

Можно писать салонам так:

> Я делаю Telegram-бота для онлайн-записи: клиент сам выбирает услугу, мастера и время, а администратор получает готовую заявку. Вот пример реализованного решения: https://emoread23.github.io/project134/

## Что можно добавить в реальный проект

- Telegram Bot API;
- база данных записей;
- админка для услуг, мастеров и графика;
- уведомления клиенту и администратору;
- интеграция с Google Calendar.

## Стек витрины

Статический HTML/CSS/JavaScript. В реальном внедрении подключается Telegram Bot API, база данных и админ-панель.
